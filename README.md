# LibraryMVC
Açıklama ve Amaç: Bu proje demo bir kütüphane otomasyon sistemidir .Net Core MVC kullanılarak inşa edilmiştir.
Model-View-Controller olmak üzere 3 katmandan oluşmaktadır. Veritabanı olarak PostgreSql kullanılmıştır.
Veritabanında tablolar oluşturulduktan sonra ide ile bağlantısı sağlandı. Sonrasında ise kullanılan ide(Visual Studio 2022)'de 1 adet controller(BookController) oluşturuldu.
Bu controller'da bulunan actionlar ile database' linq kodları ile istek atıldı,sonrasında View katmanındaki kodlarımız ile bir web-sitesi görünür kılındı.
